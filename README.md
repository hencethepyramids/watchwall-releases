# Watchwall — Releases

Official release builds and the **auto-update feed** for **[Watchwall](https://watchwall.app)** —
a fast desktop app for watching many live Twitch streams at once in one resizable grid.

This repository holds **only** the published installers and the `latest.yml` update manifest.
The application's source code lives in a separate private repository.

## ⬇️ Download

**→ [Download the latest version](https://watchwall.app/download.html) (Windows)**

Or grab it directly from the [latest release »](https://github.com/hencethepyramids/watchwall-releases/releases/latest)

> Watchwall isn't code-signed yet, so Windows SmartScreen may warn about an "unknown
> publisher." It's safe to install — click **More info → Run anyway**.

## 🔄 Auto-updates

After the first install, Watchwall keeps itself up to date automatically: on launch it checks
this repository's releases, downloads any new version in the background, and installs it with a
single click (**"Update ready — Restart"**). No manual downloads after the first one.
